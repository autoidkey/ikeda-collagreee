- 要約を生成するプログラムです．
- この要約器はスレッドを対象としています．

1. 内容物
    input_data      : 入力ディレクトリ（スレッドのcsvファイルをここに入れてください）
    output_data     : 出力ディレクトリ（要約結果のcsvファイルがここに出力されます）
    main.py         : 要約器のメインファイル
    preprocess.py   : ファイル読み書き
    anaphora.py     : 指示詞補完（KNP）
    mecab.py        : 名詞・動詞の抽出
    doc2vec.py      : doc2vecによるベクトル生成
    clusters.py     : 階層的クラスタリング
    svm.py          : 不要発言除去
    compress.py     : 文短縮（KNP）
    lexrank.py      : 重要文抽出
    README.txt      : 本ファイル

2. 入力
必ず以下のカラム名と対応するデータを含む1つのスレッドが格納されたcsvファイルを
入力ディレクトリに入れてください．
    id              : 意見ID
    title           : タイトル（スレッドの先頭の意見のみタイトルあり，返信意見はなし）
    body            : 本文
    parent_id       : 親の意見ID
    created_at      : 作成日時（フォーマット %Y-%m-%d %H:%M:%S）
    user_id         : ユーザID
注意点）作成日時は指定のフォーマットに準じてください．（年-月-日 時:分:秒）

3. 出力
要約結果が標準出力され，それを格納したcsvファイルが出力ディレクトリに出力されます．

4. 使い方
コマンドライン第1引数に入力ファイル名，第2引数に要約率を指定してください．
例）$ python main.py thread.csv 0.25
    python main.py entries.csv 0.25

5. 備考
本プログラムは，指示詞補完と文短縮でKNPを使用します．
COLLAGREEのシステムにKNPが入っていない場合は入れてください．
また，入力ディレクトリにはSVMに必要なデータと，6月に行われた実験データ
から1つのスレッドを抜き出した入力ファイル（entries.csv）が入っています．
それを参考に入力ファイルを作成してください．文字コードはUTF-8です．
